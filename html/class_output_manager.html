Detailed Description

template<typename OutType = std::wostream, typename StringType = std::wstring>
class OutputManager< OutType, StringType >

This class is used to easily format output.

Author
Dzegheim
Copyright
cc0-1.0
Date
16-02-2021
Version
0.8

OutputManager is a class used to simply manage output with various pre-built functions to format many types of printable data.

The main inspiration is the print() function from the python programming language, after which I developed some form of Stockholm syndrome.

Template Parameters
OutTypeThe type of the output stream.
StringTypeThe type of the separator and and of line strings.
Warning

Prints only the first N elements of the given ranges in rows.

Iterates over all given elements and prints the first N elements of each range in rows, range by range.

Example:

#include <vector>
#include <string>
#include "OutputManager.h"
int main () {
std::vector<int> Numbers {1, 2, 3, 4, 5};
std::vector<double> Floats {1.1, 2.1, 3.1, 4.1, 5.1};
std::vector<std::wstring> Words {L"Cat", L"Dog", L"Mouse", L"Cow", L"Salmon"};
OutputManager O;
O.FirstNElementsRows(Numbers.begin(), Numbers.end(), Floats.begin(), Words.begin());
}

Output:

1 2 3
1.1 2.1 3.1
Cat Dog Mouse

Parameters

No control is performed on the passed ranges, every iterator passed must cover a range long at least N.
